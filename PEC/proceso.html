Proceso de reciclaje del PET

El reciclaje del PET (polietileno tereftalato) es un proceso mediante el cual se recolectan, limpian y transforman botellas de plástico y otros envases para darles un nuevo uso. A continuación, se describen las etapas principales:

  1. Recolección: Se juntan las botellas de PET en el plantel escolar y en puntos de acopio.
  2. Separación: Se clasifican los residuos y se separa el PET de otros materiales.
  3. Limpieza: Las botellas se lavan para quitar etiquetas, tapas y residuos.
  4. Triturado: El PET limpio se tritura en pequeñas hojuelas o escamas.
  5. Secado: Las hojuelas se secan completamente.
  6. Reutilización: El PET reciclado se usa para fabricar nuevos productos como fibras textiles, botellas nuevas u otros objetos plásticos.
